# 🛡️ Safeguard AI  
### *“Toxic Comment Se Seedha FIR — Safeguard AI Karega Digital Justice.”*

Safeguard AI is an AI-powered cyber harassment detection platform that analyzes social media comments and automatically generates legal evidence to help victims report online abuse quickly and effectively.

The system detects toxic comments, evaluates harassment severity, and prepares structured FIR reports that can assist users in filing cybercrime complaints.

---

# 🚨 Problem

Online harassment and cyberbullying are increasing rapidly across social media platforms. Victims often face challenges such as:

- Difficulty collecting evidence
- Lack of awareness about reporting mechanisms
- Time-consuming complaint filing processes
- Emotional stress caused by repeated harassment

Safeguard AI simplifies this process by automatically detecting harassment and preparing legally usable evidence.

---

# 💡 Solution

Safeguard AI creates an automated pipeline for detecting and reporting online harassment.

The system:

1. Extracts comments from social media posts
2. Uses AI to detect toxicity and abusive language
3. Classifies harassment severity
4. Generates FIR-ready evidence reports
5. Allows users to manage cases through a dashboard

---

# ⚙️ Features

### 🔍 Social Media Comment Analysis
Scrapes and analyzes comments from social media posts.

### 🤖 AI Toxicity Detection
Uses a pretrained toxicity detection model to classify comments into:

- Low
- Medium
- High
- Vulgar

### 📊 Harassment Analytics Dashboard
Provides visual insights such as:

- Severity distribution
- Toxic comment statistics
- Harassment indicators

### 📄 FIR Report Generator
Automatically generates a structured FIR report including:

- Timestamp
- Comment text
- Toxicity score
- Severity classification
- Legal warning

### 📂 Case Management
Allows users to track harassment cases and manage generated reports.

### 📧 Email Evidence Report
Users can send harassment evidence reports directly via email.
